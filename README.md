# DevOps Digital Twin Platform

## Overview

A cloud-native platform that creates a Digital Twin of Kubernetes infrastructure to simulate deployments before production.

## Tech Stack

- React
- Material UI
- React Router
- Axios
- Recharts
- React Flow
- Spring Boot (Backend)
- PostgreSQL

## Installation

```bash
git clone https://github.com/pavanpyatla/devops-digital-twin.git

cd devops-digital-twin

npm install

npm run dev
```

## Build

```bash
npm run build
```
